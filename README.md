# Logiciel-Stock-CPP
Gestion de stock de logiciels en C++

Ce projet implémente un système simple de gestion de stock de logiciels.

Fonctionnalités :
- Ajouter un logiciel
- Modifier un logiciel
- Supprimer un logiciel
- Rechercher un logiciel
- Afficher le stock
- Sauvegarde dans un fichier
- Chargement depuis un fichier

Concepts utilisés :
- Programmation Orientée Objet (POO)
- Héritage
- Vector
- Gestion dynamique de mémoire
- Fichiers (ifstream / ofstream)
